# Debt Payoff Data

This directory contains all JSON data files for the debt payoff system.

## Files

### Core Debt Inventory
- `debt-inventory-current.json` - Current debt inventory with all accounts (PRIMARY FILE)
- `debt-inventory.json` - Backup/reference inventory

### Phase Analysis Files
Generated by the debt-payoff-orchestrator agent:

- `phase1-debt-analysis-current.json` - Debt analysis with metrics and risk assessment
- `phase2-strategy-comparison-current.json` - Comparison of avalanche, snowball, and consolidation strategies
- `phase3-payoff-roadmap-current.json` - Month-by-month payoff roadmap with milestones
- `phase4a-motivation-plan-current.json` - Psychological sustainability and motivation plan
- `phase4b-acceleration-optimizer-current.json` - Payment acceleration scenarios and optimization
- `phase4c-monitoring-protocol-current.json` - Review schedules, triggers, and contingency plans

## Usage

The dashboard (`../dashboards/debt-strategy-final-2025-11-21.html`) loads data from `debt-inventory-current.json` to dynamically populate:
- Promotional Balance Defense table
- Victory Path table with tier summaries
- Monthly payment calculations
- Total debt statistics

## Data Structure

See `debt-inventory-current.json` for the complete schema. Key sections:
- `debts[]` - Array of all debt accounts
- `validationChecks` - Totals and counts for data integrity
- Account types: credit_card, personal_loan, 401k_loan, mortgage, auto_loan, etc.
